# Tutorial

## Server

- npm init -y
"main" in package.json -> server.js

- touch server.js

- npm i express

- npm i nodemon -D

Add "start": "node server", "dev": "nodemon server" in package.json scripts

## Client

- npx create-react-app .

add "proxy": http://localhost:PORT in package.json

## DataBase (server)

- npm install mysql
